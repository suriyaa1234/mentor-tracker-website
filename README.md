##  Technology Stack
Frontend: HTML, CSS, JavaScript
Backend: JavaScript / Firebase
Database: Local Storage / Firebase
##  Description

The "Website for Mentor–Student Interaction Tracking" is a web-based application designed to improve communication between mentors and students. 

In many educational institutions, mentor–student interactions are not properly recorded or managed. This leads to difficulties in tracking student progress, feedback, and guidance. This project aims to solve that problem by providing a simple and structured platform where interactions can be recorded and viewed easily.

The website allows users to enter student details and interaction messages, which are then displayed in an organized format. This helps mentors keep track of discussions, monitor student development, and ensure effective communication.

The system is built using basic web technologies such as HTML, CSS, and JavaScript, making it lightweight and easy to use. It can be further enhanced with database integration for permanent data storage and advanced features.

Overall, this project provides an efficient and user-friendly solution for managing mentor–student interactions digitally.
